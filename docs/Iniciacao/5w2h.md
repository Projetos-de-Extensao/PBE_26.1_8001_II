## Introdução

<p align = "justify">
    O 5W2H é uma ferramenta administrativa  que pode ser utilizada para diferentes contextos dentro de uma organização, como na organização para planejamento estratégico e guia na execução de ações. O 5W2H baseia-se em 7 perguntas (What?, Why?, Where?, Who?, When?, How? e How much?), com o intuito de esclarecer e organizar ideias. A ferramenta será utilizada como um guia de ações, tornando os objetivos mais evidentes para a equipe, auxiliando na compreensão, organização e tomadas de decisões no projeto.
</p>

## Metodologia

<p align = "justify">
    A equipe se reuniu para esclarecimento sobre o tema do projeto, a partir dos pontos debatidos foi possível responder às questões do 5W2H.  
</p>


## Questões 5W2H

### Versão 1.0

### Visão geral

| What? (O que será feito?)       | **Gerência de Estágios Acadêmicos do IBMEC**                                                                                           |
| ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| Why? (Por que isso será feito?) | Para garantir o controle, a organização e a conformidade dos estágios dos alunos com as normas institucionais e educacionais.          |
| Where? (Onde será feito?)       | No ambiente institucional do IBMEC, por meio de uma plataforma digital acessível aos envolvidos.                                       |
| When? (Quando será feito?)      | De forma contínua durante os períodos letivos, acompanhando todo o ciclo dos estágios.                                                 |
| Who? (Quem irá fazer?)          | Coordenação de curso, professores orientadores e setor administrativo responsável pelos estágios.                                      |
| How? (Como será feito?)         | Por meio da utilização de um sistema de gestão de estágios que permita cadastro, acompanhamento e validação das atividades dos alunos. |
| How much? (Quanto custará?)     | Custos operacionais relacionados à manutenção do sistema e à gestão administrativa.                                                    |




### Versão 2.0

### Visão produto

| What? (O que será feito?)       | **Sistema de Gestão de Estágios Acadêmicos**                                                                                                   |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Why? (Por que isso será feito?) | Para automatizar o controle de estágios, incluindo cadastro, acompanhamento e validação das atividades dos alunos.                             |
| Where? (Onde será feito?)       | Em um sistema web acessado por navegadores, disponível para alunos, professores e coordenadores.                                               |
| When? (Quando será feito?)      | Durante o desenvolvimento do projeto, com funcionalidades entregues gradualmente.                                                              |
| Who? (Quem irá fazer?)          | Desenvolvedores do grupo e usuários finais (alunos e professores).                                                                             |
| How? (Como será feito?)         | Através do desenvolvimento de um sistema com funcionalidades como cadastro de estágio, envio de documentos, validação automática e relatórios. |
| How much? (Quanto custará?)     | Custo relacionado ao tempo de desenvolvimento, sem necessidade de investimento financeiro elevado.                                             |


## Conclusão

O 5W2H é um técnica bem simples e de fácil entendimento, sem a necessidade de um especialista. Dentro da disciplina a técnica fornece um entendimento melhor, para a equipe, sobre o projeto como responsabilidades, motivação e organização de ações.
